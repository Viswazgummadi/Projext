Image Processing

Image processing is the manipulation of an image to achieve a desired result. One common technique is edge detection, which highlights the edges in an image. In this application, we apply the Canny edge detection algorithm to the images.

Image Portion Extraction

Image portion extraction involves selecting a specific region or feature from an image. This can be useful for focusing on relevant parts of an image. In this application, we use clustering algorithms to segment the images and extract meaningful portions.

Image Classification

Image classification is the process of categorizing images into predefined classes or labels. This can be done using various algorithms and techniques. In this application, we perform multi-class classification on the extracted image portions using different classification algorithms, providing suggestions based on the results.
